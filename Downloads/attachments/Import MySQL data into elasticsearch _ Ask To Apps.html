Import MySQL data into elasticsearch

March 26, 2013 Article

There are various ways to import mysql data into elasticsearch. Couple of as follows.

By using JDBC and elasticsearch river.

Using JDBC and Elasticsearch River

Install JDBC river plugin

  1. ./bin/plugin -url http://bit.ly/10FJhEd -install river-jdbc

Download MySQL JDBC driver

  1.  
  2. http://dev.mysql.com/downloads/mirror.php?id=412177
  3.  
  4. unzipÂ mysql-connector-java-5.1.21-bin.zip
  5. cp mysql-connector-java-5.1.21-bin.jar $ES_HOME/plugins/river-jdbc/
  6. ./bin/elasticsearch -f

Import table from mysql

  1. curl -XPUT 'localhost:9200/_river/jdbc/_meta' -d '{
  2. "type" : "jdbc",
  3. "jdbc" : {
  4. "driver" : "com.mysql.jdbc.Driver",
  5. "url" : "jdbc:mysql://localhost:3306/test",
  6. "user" : "",
  7. "password" : "",
  8. "sql" : " select * from test;"
  9. },
  10. "index" : {
  11. "index" : "jdbc",
  12. "type" : "jdbc"
  13. }
  14. }'

Select table data from elasticsearch

  1. curl -XGET 'http://localhost:9200/jdbc/_search?q=*'

Display indexed data on browser interface.

  1. bin/plugin -install OlegKunitsyn/elasticsearch-browser

Open http://localhost:9200/_plugin/browser/?database=[index]&table=[type]
